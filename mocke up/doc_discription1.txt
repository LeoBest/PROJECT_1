				
					Extension(plugin) Chrome "LEARN HARD"

МЕТА?:    
ОПИС:

				Закладка №1. Пошукове вікно вікіпедія{SERCH WIKI} 

1	ADD ARCHIVE - Додати до архіву.При натискані- 
	записується пошукові результати .На компютері 
	існує папка з файлами куди записується пошукові 
	результати по кожному дню в окремі файли (назва 
	файла-дата запису) або в один загальний файл
	(з можливістю роздруку).

3 	(GOOGLE SEARCH RESULT) 2 (RESULT)- Одна кнопка.
 	При натисканні, виводить всі  результати з пошуку 
 	google і відображає їх у вікні результату(6)
 	(у випадку, якщо користувач хоче подивити всі 
 	результати  по обведеній фразі).При натисканні 
 	другий раз вдображає результат пошуку з вікіпедії 
 	по якому викликали пошукове вікно 

4 	SEARCHING...(рядок пошуку)- При виклику "LEARN HARD" 
	є можливість  ввести  пошуковий запит в рядку
 	пошуку з опцією (вибір галочкою) де саме шукати, 
 	наприклад, здійснити пошук в: 
		1. Англійській версії вікіпедії
		2. Російській версії вікіпедії 
		3. Українській версії вікіпедії
		4. У всіх версіях вікіпедії
		2. Пошук результатів в google
	і водобразити в вікні результату(6)

5 	MENU - Вибір джерел пошуку. Для пошуку (
		1. Англійській версії вікіпедії
		2. Російській версії вікіпедії 
		3. Українській версії вікіпедії
		4. У всіх версіях вікіпедії
		2. Пошук результатів в google)

6	MAIN SCREEN Головне вікно результатів пошуку. 
	Відображає результати пошуку у випадку :
	1. Введення з рядка пошуку(4)
	2. Після натискання комбінації клавіш по обведеній фразі

7	OPEN Відкрити файл в якому зберігається 
	додана користовачем інформація.

4	MAIN SCREEN Головне вікно результатів пошуку.

	1. Введення з рядка пошуку(3)

5	OPEN Відкрити файл в якому зберігається 
	додана користовачем інформація.
